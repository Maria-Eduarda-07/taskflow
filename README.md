# TaskFlow

Sistema de gerenciamento de tarefas desenvolvido em Python para aplicação dos conceitos de Engenharia de Software.

---

## Objetivo

Permitir o gerenciamento de tarefas por meio de operações CRUD (Create, Read, Update e Delete), utilizando boas práticas de desenvolvimento, testes automatizados e integração contínua.

---

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Editar tarefas
- Excluir tarefas
- Marcar tarefas como concluídas

---

## Tecnologias

- Python
- Git
- GitHub
- GitHub Actions
- Pytest

---

## Metodologia

Foi utilizada a metodologia ágil Kanban para organizar o desenvolvimento do projeto.

---

## Estrutura

```text
src/
tests/
docs/
.github/
```

---

## Mudança de Escopo

Inicialmente o projeto previa apenas um CRUD básico.

Durante o desenvolvimento foi adicionada a funcionalidade de conclusão de tarefas, permitindo acompanhar o status de cada atividade e tornando o sistema mais próximo de uma aplicação real.

---

## Como executar

```bash
python -m src.main
```

---

## Como executar os testes

```bash
pytest
```
